## About This Project

This site. Ships zero JavaScript by default — Astro's island architecture means pages are static HTML unless a component specifically needs interactivity. Tailwind CSS v4 handles styling.

### Features

- **Content Collections** — Type-safe Markdown content for projects and blog posts
- **Dark Mode** — System-aware with manual toggle
- **Responsive** — Mobile-first design that works on any screen size
- **Fast** — Zero client-side JS overhead by default

### Tech Stack

- Astro for static site generation
- Tailwind CSS v4 for styling
- Deployed on Vercel
